# Genera reportes impresionantes en Excel con Python y Pandas

###### Paso para hacer correr el proyecto;

1. PASO: Crear mi entorno virtual
   `python -m venv env `

2. PASO 2: Activar el entorno virtual ejecutando;
   `. env/Scripts/activate`

# Instalar estos Paquetes necesarios

- pip install pandas
- pip install xlsxwriter
- pip install openpyxl

##### Biblioteca o (Driver) para conectar Python con MySQL

- pip install mysql-connector-python

#### Crear/Actualizar el fichero requirements.txt:

`pip freeze > requirements.txt`

#### IMPORTANTE:

###### Para correr el proyecto solo debes ejecutar el archivo requirements.txt obvio debes haber importado la tabla en tu gestor de BD y haber configurado la conexi√≥n, por ultimo ejecutas `pip install -r requirements.txt` ya estando dentro de tu entorno virtual.
